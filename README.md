# Lightweight and Simple Web API
The lightweight and simple implementation of build Web API/Web Service Framework using PHP Native.

This framework implements MVC design pattern and contains small size of source.

It comes with simple template engine that you can use to separate your layout/master page and your content page.
# Installation

clone this repo
> git clone https://github.com/galihanggara68/apiframework.git

run on your favorite server that support PHP
